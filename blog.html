What are the differences between map, forEach, filter and find?
  • .map(), executes the same code on every element in an array and returns a new array with the updated elements.
  • .forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.
  • .filter(), checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.
  • .find(), checks every element in an array until it meets a certain criteria and returns a first element that return truthy for the criteria.
What are the differences between Local Storage, Session Storage and Cookies?
  • Local storage allows for larger amounts of data to be stored, but the data is not deleted when the browser is closed.
  • In Session storage, the data is only stored for the current session. This means that the data will be deleted when the user closes the browser.
  • Cookies are limited to 4KB of data and are often used to store data that is not sensitive, such as user preferences.
How does the Event Loop works in JavaScript?

The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.

How does the chooseQuiz function work?

chooseQuiz function actually selects an answer from multiple options. And this function gets two parameters, one is which question, identified by index number and second one is user given answer. Then if specific quiz item doesn't belong to answer variable, quiz item adds in answer variable with user given answer. Or if already specific quiz item belongs to answer variable, then replaces current item by new item in answer variable which is checked by id. And finally, display the quiz item in answers container.
